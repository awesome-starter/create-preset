## Features

Provides the ability to quickly create preset projects.

## Usage

With NPM:

```bash
npm init preset@latest init
```

With Yarn:

```bash
yarn create preset init
```

With PNPM:

```bash
pnpm create preset init
```

Then follow the prompts!
